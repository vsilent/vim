pcntl_wstopsig

(PHP 4 >= 4.0.7, PHP 5)

pcntl_wstopsig — Returns the signal which caused the child to stop

Описание

int pcntl_wstopsig ( int $status )

Returns the number of the signal which caused the child to stop. This function is only useful if pcntl_wifstopped() returned TRUE.

Список параметров

status

Параметр status - это переменная, переданная по ссылке в функцию pcntl_waitpid().

Возвращаемые значения

Returns the signal number.
