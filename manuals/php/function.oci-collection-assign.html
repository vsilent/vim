collection->assign — Присваивает коллекции значение другой уже существующей коллекции

Описание

bool collection->assign ( object $from )

Присваивает коллекции значение другой, ранее созданной коллекции. Обе коллекции должны быть созданы ранее с помощью oci_new_collection().

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
