Установка

Чтобы включить поддержку GNU gettext в сборку PHP, добавьте в конфигурацию ключ --with-gettext[=DIR], где DIR - папка установки gettext, обычно это /usr.

Замечание: Примечание для тех, кто использует Win32 Скопируйте библиотеку gnu_gettext.dll из папки DLL пакета PHP/Win32 в папку SYSTEM32. (например: C:\WINNT\SYSTEM32 или C:\WINDOWS\SYSTEM32). Начиная с PHP 4.2.3 имя изменилось на libintl-1.dll, также нужно скопировать iconv.dll. libintl-1.dll не требуется начиная с PHP 4.3.8, iconv.dll не требуется начиная с PHP 5.
