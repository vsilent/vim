key

(PHP 4, PHP 5)

key — Выбрать ключ из ассоциативного массива

Описание

mixed key ( array &$array )

key() возвращает индекс текущего элемента массива.

Пример #1 Пример использования key()

<?php
$array = array(
    'fruit1' => 'apple',
    'fruit2' => 'orange',
    'fruit3' => 'grape',
    'fruit4' => 'apple',
    'fruit5' => 'apple');

// этот цикл выведет все ключи ассоциативного массива,
// значения которых равны "apple"
while ($fruit_name = current($array)) {
    if ($fruit_name == 'apple') {
        echo key($array).'<br />';
    }
    next($array);
}
?>

См. также current() и next().
