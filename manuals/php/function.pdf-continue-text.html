PDF_continue_text

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_continue_text — Output text in next line

Описание

bool PDF_continue_text ( resource $p , string $text )

Prints text at the next line. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
