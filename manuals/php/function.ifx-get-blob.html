ifx_get_blob — Return the content of a blob object

Описание

string ifx_get_blob ( int $bid )

Returns the content of the blob object.

Список параметров

bid

The BLOB object id.

Возвращаемые значения

The contents of the BLOB as a string, or FALSE on errors.
