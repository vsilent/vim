date_parse

(PHP 5 >= 5.1.3)

date_parse — Returns associative array with detailed info about given date

Описание

array date_parse ( string $date )

Список параметров

date

Date in format accepted by strtotime().

Возвращаемые значения

Returns array with information about the parsed date on success, or FALSE on failure.

Errors/Exceptions

In case the date format has an error, the element 'errors' will contains the error messages.

Примеры

Пример #1 A date_parse() example

<?php
print_r(date_parse("2006-12-12 10:00:00.5"));
?>

Результат выполнения данного примера:

Array
(
    [year] => 2006
    [month] => 12
    [day] => 12
    [hour] => 10
    [minute] => 0
    [second] => 0
    [fraction] => 0.5
    [warning_count] => 0
    [warnings] => Array()
    [error_count] => 0
    [errors] => Array()
    [is_localtime] => 
)
