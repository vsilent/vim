$HTTP_GET_VARS [deprecated] — HTTP GET variables

Описание

An associative array of variables passed to the current script via the HTTP GET method.

$HTTP_GET_VARS contains the same initial information, but is not a superglobal. (Note that $HTTP_GET_VARS and $_GET are different variables and that PHP handles them as such)

Список изменений

Версия Описание
4.1.0 Introduced $_GET that deprecated $HTTP_GET_VARS.

Примеры

Пример #1 $_GET example

<?php
echo 'Hello ' . htmlspecialchars($_GET["name"]) . '!';
?>

Assuming the user entered http://example.com/?name=Hannes

Результатом выполнения данного примера будет что-то подобное:

Hello Hannes!

Примечания

Замечание: This is a 'superglobal', or automatic global, variable. This simply means that it is available in all scopes throughout a script. There is no need to do global $variable; to access it within functions or methods.
