readline_redisplay

(PHP 5 >= 5.1.0)

readline_redisplay — Redraws the display

Описание

void readline_redisplay ( void )

Redraws readline to redraw the display.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
