hw_Error

(PHP 4)

hw_Error — Error number

Описание

int hw_error ( int $connection )

Returns the last error number, for the last command.

Список параметров

connection

The connection identifier.

Возвращаемые значения

Returns the last error number or 0 if no error occured.
