ctype_print

(PHP 4 >= 4.0.4, PHP 5)

ctype_print — Проверяет, является ли указанный символ печатаемым

Описание

bool ctype_print ( string $text )

Проверяет, все ли символы в переданной строке text являются печатаемыми.

Список параметров

text

Проверяемая строка.

Возвращаемые значения

Возвращает TRUE, если каждый символ в строке text фактически создает вывод (включая пустой). Возвращает FALSE, если text содержит управляющие символы, или символы, которые вобще не имеют никакого вывода или функции управления.

Примеры

Пример #1 Пример использования ctype_print()

<?php
$strings = array('string1' => "asdf\n\r\t", 'string2' => 'arf12', 'string3' => 'LKA#@%.54');
foreach ($strings as $name => $testcase) {
    if (ctype_print($testcase)) {
        echo "Строка '$name' состоит только из печатаемых символов.\n";
    } else {
        echo "Строка '$name' не состоит только из печатаемых символов.\n";
    }
}
?>

Результат выполнения данного примера:

Строка 'string1' не состоит только из печатаемых символов.
Строка 'string2' состоит только из печатаемых символов.
Строка 'string3' состоит только из печатаемых символов.
