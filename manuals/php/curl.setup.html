Требования

Для использования функций CURL необходимо установить пакет » CURL. Необходимо мспользовать версию CURL 7.0.2-beta или старше, с более младшими версиями PHP не работает. Для PHP 4.2.3 необходимо использовать версию CURL 7.9.0 или старше. Начиная PHP 4.3.0, необходима CURL 7.9.8 или старше. PHP 5.0.0 требует CURL не младше 7.10.5.
