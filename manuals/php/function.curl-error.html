curl_error

(PHP 4 >= 4.0.3, PHP 5)

curl_error — Возвращает строку с описанием последней ошибки

Описание

string curl_error ( resource $ch )

Возвращает сообщение об ошибке для последней операции cURL с дескриптором ch , или '' (пустую строку), если ошибки не было.

См. также описание функции curl_errno() и » коды ошибок CURL.
