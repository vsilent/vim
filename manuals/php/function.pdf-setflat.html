PDF_setflat

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_setflat — Set flatness

Описание

bool PDF_setflat ( resource $pdfdoc , float $flatness )

Sets the flatness parameter. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
