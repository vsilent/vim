Введение

Эти функции позволяют вам работать с Oracle версий 10/9/8с помощью библиотеки Oracle Call Interface (OCI). Они поддерживают привязку переменных PHP к соответствующим меткам Oracle, расширенную поддержку LOB, FILE и ROWID.
