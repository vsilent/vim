PDF_makespotcolor

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_makespotcolor — Make spot color

Описание

int PDF_makespotcolor ( resource $p , string $spotname )

Finds a built-in spot color name, or makes a named spot color from the current fill color. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
