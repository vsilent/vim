exit

(PHP 4, PHP 5)

exit — Output a message and terminate the current script

Описание

void exit ([ string $status ] )
void exit ( int $status )

Terminates execution of the script.

Список параметров

status

If status is a string, this function prints the status just before exiting.

If status is an integer, that value will also be used as the exit status. Exit statuses should be in the range 0 to 254, the exit status 255 is reserved by PHP and shall not be used. The status 0 is used to terminate the program successfully.

Замечание: PHP >= 4.2.0 does NOT print the status if it is an integer.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 exit() example

<?php

$filename = '/path/to/data-file';
$file = fopen($filename, 'r')
    or exit("unable to open file ($filename)");

?>

Пример #2 exit() status example

<?php

//exit program normally
exit;
exit();
exit(0);

//exit with an error code
exit(1);
exit(0376); //octal

?>

Примечания

Замечание: Поскольку это языковая конструкция, а не функция, она не может вызываться при помощи переменных функций

Замечание: This language construct is equivalent to die().
