PDF_close_pdi_page

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_close_pdi_page — Close the page handle

Описание

bool PDF_close_pdi_page ( resource $p , int $page )

Closes the page handle, and frees all page-related resources. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
