(No version information available, might be only in CVS)

DOMDocument::createDocumentFragment — Create new document fragment

Описание

DOMDocumentFragment DOMDocument::createDocumentFragment ( void )

This function creates a new instance of class DOMDocumentFragment. Этот узел не будет отображаться в документе до тех пор, пока он не будет вставлен, к пример функцией DOMNode->appendChild().

Возвращаемые значения

The new DOMDocumentFragment or FALSE if an error occured.
