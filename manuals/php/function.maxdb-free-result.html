maxdb_free_result

result->free

(No version information available, might be only in CVS)

result->free — Frees the memory associated with a result

Описание

Procedural style:

void maxdb_free_result ( resource $result )

Object oriented style (method):

result
void free ( void )

The maxdb_free_result() function frees the memory associated with the result represented by the result parameter, which was allocated by maxdb_query(), maxdb_store_result() or maxdb_use_result().

Замечание: You should always free your result with maxdb_free_result(), when your result resource is not needed anymore.

Возвращаемые значения

This function doesn't return any value.
