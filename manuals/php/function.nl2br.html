nl2br

(PHP 4, PHP 5)

nl2br — Вставляет HTML код разрыва строки перед каждым переводом строки

Description

string nl2br ( string $string )

Возвращает строку string , в которой перед каждым переводом строки вставлен '<br />'

Замечание: Начиная с PHP 4.0.5, nl2br() работает в соответствии со стандартом XHTML. До версии 4.0.5 использовалось '<br>' вместо '<br />'.

Пример #1 Пример использования nl2br()

<?php
echo nl2br("foo - это вам не\n bar");
?>

выведет: 

foo - этовам не<br />
 bar

См. также описание функций htmlspecialchars(), htmlentities(), wordwrap() и str_replace().
