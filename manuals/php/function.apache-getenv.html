apache_getenv

(PHP 4 >= 4.3.0, PHP 5)

apache_getenv — Получить переменную окружения подпроцесса сервера Apache

Описание

string apache_getenv ( string $variable [, bool $walk_to_top ] )

Получить переменную окружения сервера Apache, указанную параметром variable .

Эта функция требует Apache 2, в противном случае она неопределена.

Список параметров

variable

Переменная окружения сервера Apache

walk_to_top

Пробовать ли получить переменную вержнего уровня, доступную для всех уровней сервера Apache.

Возвращаемые значения

Значение переменной окружения сервера Apache в случае успеха или FALSE при неудаче.

Примеры

Пример #1 Пример использования apache_getenv()

Приведенный ниже пример показывает, как можно получить значение переменной окружения сервера Apache SERVER_ADDR.

<?php
$ret = apache_getenv("SERVER_ADDR");
echo $ret;
?>

Результатом выполнения данного примера будет что-то подобное:

42.24.42.240
