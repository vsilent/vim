PDF_set_value

(PHP 4 >= 4.0.1, PECL pdflib:1.0-2.1.3)

PDF_set_value — Set numerical parameter

Описание

bool PDF_set_value ( resource $p , string $key , float $value )

Sets the value of some PDFlib parameter with numerical type. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
