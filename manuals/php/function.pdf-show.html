PDF_show

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_show — Output text at current position

Описание

bool PDF_show ( resource $pdfdoc , string $text )

Prints text in the current font and size at the current position. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
