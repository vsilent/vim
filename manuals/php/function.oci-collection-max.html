collection->max

(No version information available, might be only in CVS)

collection->max — Возвращает максимальное количество элементов в коллекции.

Описание

int collection->max ( void )

Возвращает максимальное количество элементов в коллекции. Для коллекций типа VARRAY - это максимальная длина массива. Если возвращаемое значение равно нулю, то количество элементов в коллекции не ограничено.

См. также oci_collection_size().
