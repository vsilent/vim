PDF_add_thumbnail

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_add_thumbnail — Add thumbnail for current page

Описание

bool PDF_add_thumbnail ( resource $pdfdoc , int $image )

Adds an existing image as thumbnail for the current page. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
