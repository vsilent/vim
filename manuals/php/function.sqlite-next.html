sqlite_next

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_next — Переходит на следующую запись

Описание

bool sqlite_next ( resource $result )

sqlite_next() перемещает внутренний указатель результата запроса result на следующую запись. Возвращает FALSE если указатель находился на последней записи перед вызовом функции, иначе возвращает TRUE.

Замечание: Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.
