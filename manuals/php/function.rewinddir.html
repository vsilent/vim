rewinddir

(PHP 4, PHP 5)

rewinddir — Сбросить дескриптор каталога

Описание

void rewinddir ( resource $dir_handle )

Сбрасывает поток каталога, переданный в параметре дескриптор_каталога таким образом, чтобы тот указывал на начало каталога.
