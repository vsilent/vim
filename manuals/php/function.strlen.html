strlen

(PHP 4, PHP 5)

strlen — Возвращает длину строки

Описание

int strlen ( string $string )

Возвращает длину строки string .

Пример #1 Пример использования strlen()

<?php
$str = 'abcdef';
echo strlen($str); // 6

$str = ' ab cd ';
echo strlen($str); // 7
?>

См. также описание функций count() и mb_strlen().
