tan

(PHP 4, PHP 5)

tan — Tangent

Описание

float tan ( float $arg )

tan() returns the tangent of the arg parameter. The arg parameter is in radians.

Список параметров

arg

The argument to process in radians

Возвращаемые значения

The tangent of arg

Примеры

Пример #1 tan() example

<?php

echo tan(M_PI_4); // 1

?>
