PDF_begin_layer

(PECL pdflib:2.0-2.1.3)

PDF_begin_layer — Start layer

Описание

bool PDF_begin_layer ( resource $pdfdoc , int $layer )

Starts a layer for subsequent output on the page. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

This function requires PDF 1.5.
