bzflush

(PHP 4 >= 4.3.3, PHP 5, PECL bz2:1.0)

bzflush — Вызывает запись всех буфферизированных данных

Описание

integer bzflush ( resource $bz )

Записывает все буфферизированные bzip2 данные в файл, на который ссылается указатель bz .

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

См. также bzread() и bzwrite().
