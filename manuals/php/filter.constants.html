Предопределенные константы

Перечисленные ниже константы определены данным расширением и могут быть доступны только в том случае, если PHP был собран с поддержкой этого расширения или же в том случае, если данное расширение подгружается во время выполнения.

INPUT_POST (integer)
POST переменные.
INPUT_GET (integer)
GET переменные.
INPUT_COOKIE (integer)
COOKIE переменные.
INPUT_ENV (integer)
ENV переменные.
INPUT_SERVER (integer)
SERVER переменные.
INPUT_SESSION (integer)
SESSION переменные. (не реализовано)
INPUT_REQUEST (integer)
REQUEST переменные. (не реализовано)
FILTER_FLAG_NONE (integer)
Не использовать флаги.
FILTER_REQUIRE_SCALAR (integer)
Флаг используется для указания необходимости скаларного значения в качестве входного значения.
FILTER_REQUIRE_ARRAY (integer)
Требовать массив в качестве входного значения.
FILTER_FORCE_ARRAY (integer)
Всегда возвращать массив.
FILTER_NULL_ON_FAILURE (integer)
Использовать NULL вместо FALSE в случае ошибки.
FILTER_VALIDATE_INT (integer)
ID фильтра "int".
FILTER_VALIDATE_BOOLEAN (integer)
ID фильтра "boolean".
FILTER_VALIDATE_FLOAT (integer)
ID фильтра "float".
FILTER_VALIDATE_REGEXP (integer)
ID фильтра "validate_regexp".
FILTER_VALIDATE_URL (integer)
ID фильтра "validate_url".
FILTER_VALIDATE_EMAIL (integer)
ID фильтра "validate_email".
FILTER_VALIDATE_IP (integer)
ID фильтра "validate_ip".
FILTER_DEFAULT (integer)
ID фильтра по умолчанию ("string").
FILTER_UNSAFE_RAW (integer)
ID фильтра "unsafe_raw".
FILTER_SANITIZE_STRING (integer)
ID фильтра "string".
FILTER_SANITIZE_STRIPPED (integer)
ID фильтра "stripped".
FILTER_SANITIZE_ENCODED (integer)
ID фильтра "encoded".
FILTER_SANITIZE_SPECIAL_CHARS (integer)
ID фильтра "special_chars".
FILTER_SANITIZE_EMAIL (integer)
ID фильтра "email".
FILTER_SANITIZE_URL (integer)
ID фильтра "url".
FILTER_SANITIZE_NUMBER_INT (integer)
ID фильтра "number_int".
FILTER_SANITIZE_NUMBER_FLOAT (integer)
ID фильтра "number_float".
FILTER_SANITIZE_MAGIC_QUOTES (integer)
ID фильтра "magic_quotes".
FILTER_CALLBACK (integer)
ID фильтра "callback".
FILTER_FLAG_ALLOW_OCTAL (integer)
Разрешить восьмеричную запись (0[0-7]+) в фильтре "int".
FILTER_FLAG_ALLOW_HEX (integer)
Разрешить шестнадцатиричную запись (0x[0-9a-fA-F]+) в фильтре "int".
FILTER_FLAG_STRIP_LOW (integer)
Удалять символы с ASCII кодом, меньшим 32.
FILTER_FLAG_STRIP_HIGH (integer)
Удалять символы с ASCII кодом, большим 127.
FILTER_FLAG_ENCODE_LOW (integer)
Кодировать символы с ASCII кодом, меньшим 32.
FILTER_FLAG_ENCODE_HIGH (integer)
Кодировать символы с ASCII кодом, большим 127.
FILTER_FLAG_ENCODE_AMP (integer)
Кодировать &.
FILTER_FLAG_NO_ENCODE_QUOTES (integer)
Не кодировать ' и ".
FILTER_FLAG_EMPTY_STRING_NULL (integer)
(На данный момент не используется.)
FILTER_FLAG_ALLOW_FRACTION (integer)
Разрешить дробную часть в фильтре "number_float".
FILTER_FLAG_ALLOW_THOUSAND (integer)
Разрешить разделитель (,) между группами цифр по 3 в фильтре "number_float".
FILTER_FLAG_ALLOW_SCIENTIFIC (integer)
Разрешить научную запись (e, E) в фильтре "number_float".
FILTER_FLAG_SCHEME_REQUIRED (integer)
Требовать наличие составляющей "схема" в фильтре "validate_url".
FILTER_FLAG_HOST_REQUIRED (integer)
Требовать наличие составляющей "хост" в фильтре "validate_url".
FILTER_FLAG_PATH_REQUIRED (integer)
Требовать наличие составляющей "путь" в фильтре "validate_url".
FILTER_FLAG_QUERY_REQUIRED (integer)
Требовать наличие строки запроса в фильтре "validate_url".
FILTER_FLAG_IPV4 (integer)
Разрешить только IPv4 адреса в фильтре "validate_ip".
FILTER_FLAG_IPV6 (integer)
Разрешить только IPv6 адреса в фильтре "validate_ip".
FILTER_FLAG_NO_RES_RANGE (integer)
Запретить зарезервированные адреса в фильтре "validate_ip".
FILTER_FLAG_NO_PRIV_RANGE (integer)
Запретить частные адреса в фильтре "validate_ip".
