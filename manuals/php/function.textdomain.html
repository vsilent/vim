textdomain

(PHP 4, PHP 5)

textdomain — устанавливает каталог сообщений по уммолчанию

Описание

string textdomain ( string $text_domain )

Устанавливает каталог сообщений, в котором будет искаться перевод строки, переданной функции gettext(), обычно называется имя_приложения.mo. Возвращается каталог сообщений, установленный по умолчанию до этого. Вызовите эту функцию с аргументом NULL, чтобы получить имя текущего каталога сообщений не меняя его.
