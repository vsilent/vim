swf_rotate

(PHP 4)

swf_rotate — Rotate the current transformation

Описание

void swf_rotate ( float $angle , string $axis )

Rotates the current transformation by a given angle around the given axis .

Список параметров

angle

The rotation angle.

axis

The axis. Valid values axis are x (the x axis), y (the y axis) or z (the z axis).

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
