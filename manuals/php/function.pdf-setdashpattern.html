PDF_setdashpattern

(PECL pdflib:2.0-2.1.3)

PDF_setdashpattern — Set dash pattern

Описание

bool PDF_setdashpattern ( resource $pdfdoc , string $optlist )

Sets a dash pattern defined by an option list. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
