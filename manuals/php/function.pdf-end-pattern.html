PDF_end_pattern

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_end_pattern — Finish pattern

Описание

bool PDF_end_pattern ( resource $p )

Finishes the pattern definition. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
