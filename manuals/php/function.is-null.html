is_null

(PHP 4 >= 4.0.4, PHP 5)

is_null — Finds whether a variable is NULL

Описание

bool is_null ( mixed $var )

Finds whether the given variable is NULL.

Список параметров

var

The variable being evaluated.

Возвращаемые значения

Returns TRUE if var is null, FALSE otherwise.
