swf_textwidth

(PHP 4)

swf_textwidth — Get the width of a string

Описание

float swf_textwidth ( string $str )

Gives the width of the string in pixels, using the current font and font size.

Список параметров

str

The string.

Возвращаемые значения

Returns the line width, as a float.
