HttpResponse::setBufferSize

(PECL pecl_http:0.10.0-1.5.5)

HttpResponse::setBufferSize — Set buffer size

Описание

static bool HttpResponse::setBufferSize ( int $bytes )

Sets the send buffer size of the throttling mechanism.

Замечание: Provides a basic throttling mechanism, which will yield the current process or thread until the entity has been completely sent.

Замечание: This may not work as expected with the following SAPI(s): FastCGI.

Список параметров

bytes

the chunk size in bytes

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
