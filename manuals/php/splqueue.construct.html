(No version information available, might be only in CVS)

SplQueue::__construct — Constructs a new queue implemented using a doubly linked list

Описание

SplQueue::__construct ( void )

This constructs a new empty queue.

Замечание: This method automatically sets the iterator mode to SplDoublyLinkedList::IT_MODE_FIFO.

Список параметров

This function has no parameters.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 SplQueue::__construct() example

<?php
$q = new SplQueue();

$q[] = 1;
$q[] = 2;
$q[] = 3;

foreach ($q as $elem)  {
 echo $elem."\n";
}
?>
