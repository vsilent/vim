yp_errno

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5)

yp_errno — Returns the error code of the previous operation

Описание

int yp_errno ( void )

Returns the error code of the previous operation.

Возвращаемые значения

Returns one of the YPERR_XXX error constants.
