sqlite_num_rows

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_num_rows — Возвращает количество записей в результате запроса

Описание

int sqlite_num_rows ( resource $result )

Возвращает количество записей в результате запроса result .

Замечание: Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.
