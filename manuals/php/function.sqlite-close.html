sqlite_close

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_close — Закрывает базу данных SQLite

Описание

void sqlite_close ( resource $dbhandle )

Закрывает базу данных, идентификатор соединения с которой передан через аргумент dbhandle . Если соединение с базой данных было постоянным, также удаляет это соединение из списка постоянных.
