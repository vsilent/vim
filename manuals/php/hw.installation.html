Установка

Это расширение » PECL не поставляется вместе с PHP.

In order to use these functions you must compile PHP with Hyperwave support by using the --with-hyperwave[=DIR] configure option.

Windows users will enable php_hyperwave.dll inside of php.ini in order to use these functions. Вы можете скачать DLL этого расширения PECL со страницы » PHP Downloads или » http://snaps.php.net/.
