PDF_circle

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_circle — Draw a circle

Описание

bool PDF_circle ( resource $pdfdoc , float $x , float $y , float $r )

Adds a circle. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
