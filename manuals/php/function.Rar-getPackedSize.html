(No version information available, might be only in CVS)

Rar::getPackedSize — Возвращает размер сжатого элемента

Описание

int Rar::getPackedSize ( void )

Возвращает размер сжатого элемента

Пример #1 Rar::() пример использования

<?php

$rar_file = rar_open('example.rar') or die("Невозможно открыть Rar архив");

$entry = rar_entry_get($rar_file, 'Dir/file.txt') or die("Такого элемента в архиве не существует");

echo "Размер сжатого элемента " . $entry->getName() . " = " . $entry->getPackedSize() . " bytes";

?>

Rar::getPackedSize() возвращает FALSE при ошибке.
