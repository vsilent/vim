cosh

(PHP 4 >= 4.0.7, PHP 5)

cosh — Hyperbolic cosine

Описание

float cosh ( float $arg )

Returns the hyperbolic cosine of arg , defined as (exp(arg) + exp(-arg))/2.

Список параметров

arg

The argument to process

Возвращаемые значения

The hyperbolic cosine of arg
