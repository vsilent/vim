rar_entry_get — Возвращает объект из Rar архива

Описание

RarEntry rar_entry_get ( resource $rar_file , string $entry_name )

Возвращает объект из Rar архива.

Пример #1 Rar::() пример

<?php

$rar_file = rar_open('example.rar') or die("Не удалось открыть Rar архив");

$entry = rar_entry_get($rar_file, 'Dir/file.txt') or die("Такого элемента в архиве не существует");

print_r($entry);

?>

rar_get_entry() возвращает объект или FALSE при ошибке.
