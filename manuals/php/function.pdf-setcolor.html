PDF_setcolor

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_setcolor — Set fill and stroke color

Описание

bool PDF_setcolor ( resource $p , string $fstype , string $colorspace , float $c1 , float $c2 , float $c3 , float $c4 )

Sets the current color space and color. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
