gztell

(PHP 4, PHP 5)

gztell — Возвращает текущую позицию чтения/записи в gz-файле

Описание

int gztell ( resource $zp )

Возвращает позицию в файле по его указателю, т.е. смещение в потоке данного файла.

Список параметров

zp

Указатель на gz-файл, возвращённый, после его успешного открытия, функцией gzopen().

Возвращаемые значения

Текущая позиция или FALSE в случае ошибки.
