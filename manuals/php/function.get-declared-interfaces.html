get_declared_interfaces

(PHP 5)

get_declared_interfaces — Возвращает массив всех объявленных в текущей итерации интерфейсов.

Описание

array get_declared_interfaces ( void )

Функция возвращает массив имен объявленных в текущем скрипте интерфейсов.

Пример #1 Пример использования get_declared_interfaces()

<?php
print_r(get_declared_interfaces());
?>

Пример выведет что-то близкое к: 

Array
(
    [0] => Traversable
    [1] => IteratorAggregate
    [2] => Iterator
    [3] => ArrayAccess
    [4] => reflector
    [5] => RecursiveIterator
    [6] => SeekableIterator
)
