cos

(PHP 4, PHP 5)

cos — Cosine

Описание

float cos ( float $arg )

cos() returns the cosine of the arg parameter. The arg parameter is in radians.

Список параметров

arg

An angle in radians

Возвращаемые значения

The cosine of arg

Примеры

Пример #1 cos() example

<?php

echo cos(M_PI); // -1

?>
