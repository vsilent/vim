oci_server_version

(PHP 5, PECL oci8:1.1-1.2.4)

oci_server_version — Возвращает строку с информацией о версии сервера Oracle

Описание

string oci_server_version ( resource $connection )

Возвращает строку с информацией о версии сервера Oracle, с которым установлено соединение connection или FALSE в случае ошибки.

Пример #1 Пример использования oci_server_version()

<?php
    $conn = oci_connect("scott", "tiger");
    echo "Server Version: " . oci_server_version($conn);
    oci_close($conn);
?>

Замечание: В версиях PHP ниже 5.0.0 эта функция называлась ociserverversion(). В PHP 5.0.0 и выше ociserverversion() является алиасом oci_server_version(), поэтому вы можете продолжать использовать это имя, однако это не рекомендуется.
