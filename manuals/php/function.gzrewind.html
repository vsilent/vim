gzrewind

(PHP 4, PHP 5)

gzrewind — Перемещает индикатор позиции в gz-файле в начало

Описание

bool gzrewind ( resource $zp )

Устанавливает индикатор позиции файла на начало потока этого файла.

Список параметров

zp

Указатель на gz-файл, возвращённый, после его успешного открытия, функцией gzopen().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
