PDF_setmatrix

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_setmatrix — Set current transformation matrix

Описание

bool PDF_setmatrix ( resource $p , float $a , float $b , float $c , float $d , float $e , float $f )

Explicitly sets the current transformation matrix. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
