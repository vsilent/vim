PDF_set_layer_dependency

(PECL pdflib:2.0-2.1.3)

PDF_set_layer_dependency — Define relationships among layers

Описание

bool PDF_set_layer_dependency ( resource $pdfdoc , string $type , string $optlist )

Defines hierarchical and group relationships among layers. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

This function requires PDF 1.5.
