Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Параметры конфигурации Filter
Имя Значение по умолчанию Область изменения Changelog
filter.default "unsafe_raw" PHP_INI_PERDIR PHP_INI_ALL в filter <= 0.9.4. Доступна с PHP 5.2.0.
filter.default_flags NULL PHP_INI_PERDIR PHP_INI_ALL в filter <= 0.9.4. Доступна с PHP 5.2.0.
Для подробного описания констант PHP_INI_*, обратитесь к документации функции ini_set().

Краткое разъяснение конфигурационных директив.

filter.default string

Фильтрует все $_GET, $_POST, $_COOKIE и $_REQUEST данные этим фильтром. Исходные данные могут быть получены с помощью filter_input().

Принимает имя указанного фильтра как значение по умолчанию. См. список существующих фильтров для получения списка имен фильтров.

filter.default_flags integer
