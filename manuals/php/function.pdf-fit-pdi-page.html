PDF_fit_pdi_page

(PECL pdflib:2.0-2.1.3)

PDF_fit_pdi_page — Place imported PDF page

Описание

bool PDF_fit_pdi_page ( resource $pdfdoc , int $page , float $x , float $y , string $optlist )

Places an imported PDF page on the page, subject to various options. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
