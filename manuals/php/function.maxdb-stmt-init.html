maxdb_stmt_init

maxdb->stmt_init

(PECL maxdb:1.0-7.6.00.38)

maxdb->stmt_init — Initializes a statement and returns an resource for use with maxdb_stmt_prepare

Описание

Procedural style :

resource maxdb_stmt_init ( resource $link )

Object oriented style (property):

maxdb
object stmt_init ( void )

Allocates and initializes a statement resource suitable for maxdb_stmt_prepare().

Замечание: Any subsequent calls to any maxdb_stmt function will fail until maxdb_stmt_prepare() was called.

Возвращаемые значения

Returns an resource.
