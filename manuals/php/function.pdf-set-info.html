PDF_set_info

(PHP 4 >= 4.0.1, PECL pdflib:1.0-2.1.3)

PDF_set_info — Fill document info field

Описание

bool PDF_set_info ( resource $p , string $key , string $value )

Fill document information field key with value . Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
