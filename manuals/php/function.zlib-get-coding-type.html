zlib_get_coding_type

(PHP 4 >= 4.3.2, PHP 5)

zlib_get_coding_type — Возвращает тип кодирования, используемый для сжатия вывода

Описание

string zlib_get_coding_type ( void )

Возвращает тип кодирования, используемый для сжатия вывода.

Возвращаемые значения

Возможные значения: gzip, deflate или FALSE.
