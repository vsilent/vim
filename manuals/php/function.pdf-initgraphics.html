PDF_initgraphics

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_initgraphics — Reset graphic state

Описание

bool PDF_initgraphics ( resource $p )

Reset all color and graphics state parameters to their defaults. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
