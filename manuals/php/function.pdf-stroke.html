PDF_stroke

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_stroke — Stroke path

Описание

bool PDF_stroke ( resource $p )

Strokes the path with the current color and line width, and clear it. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
