Установка

This » PECL extension is not bundled with PHP.

Windows users must enable php_printer.dll inside of php.ini in order to use these functions. Вы можете получить не встроенное расширение PECL с различных страниц снимков PECL (выберите соответствующий репозиторий для вашей версии PHP): » PECL для PHP 4.3.x, » PECL для PHP 5.0.x или » PECL Нестабильные.
