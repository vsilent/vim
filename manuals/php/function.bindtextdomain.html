bindtextdomain

(PHP 4, PHP 5)

bindtextdomain — Указать путь к каталогу сообщений

Описание

string bindtextdomain ( string $domain , string $directory )

Возвращает текущий путь.

Пример #1 Пример

<?php

$domain = 'myapp';
echo bindtextdomain($domain, '/usr/share/myapp/locale'); 

?>

Вывод: 

/usr/share/myapp/locale
