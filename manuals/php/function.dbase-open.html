dbase_open

(PHP 4, PHP 5)

dbase_open — Opens a database

Описание

int dbase_open ( string $filename , int $mode )

dbase_open() opens a dBase database with the given access mode.

Замечание: Когда опция safe mode включена, PHP проверяет, имеют ли файлы/каталоги, с которыми вы собираетесь работать, такой же UID (владельца), как и выполняемый скрипт.

Замечание: На поведение этой функции влияет значение директивы open_basedir.

Список параметров

filename

The name of the database. It can be a relative or absolute path to the file where dBase will store your data.

mode

An integer which correspond to those for the open() system call (Typically 0 means read-only, 1 means write-only, and 2 means read and write).

Замечание: You can't open a dBase file in write-only mode as the function will fail to read the headers information and thus you can't use 1 as mode .

Примеры

Пример #1 Opening a dBase database file

<?php

// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  // read some data ..
  
  dbase_close($db);
}

?>

Возвращаемые значения

Returns a database link identifier if the database is successfully opened, or FALSE if an error occurred.
