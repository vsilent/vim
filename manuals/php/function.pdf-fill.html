PDF_fill

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_fill — Fill current path

Описание

bool PDF_fill ( resource $p )

Fills the interior of the current path with the current fill color. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
