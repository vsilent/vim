getrandmax

(PHP 4, PHP 5)

getrandmax — Вовзращает максимально возможное случайное число

Описание

int getrandmax ( void )

Возвращает максимальное значение, которое можно получить функцией rand().

См. также rand(), srand() и mt_getrandmax().
