class_exists

(PHP 4, PHP 5)

class_exists — Проверяет был ли объявлен класс

Описание

bool class_exists ( string $class_name )

Функция возвращает TRUE если классclass_name был объявлен. В противном случае функция возвращает FALSE.

См. также get_declared_classes().
