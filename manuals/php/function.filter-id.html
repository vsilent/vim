filter_id

(PHP 5 >= 5.2.0, PECL filter:0.11.0)

filter_id — Возвращает ID фильтра, принадлежащий именованному фильтру

Описание

int filter_id ( string $filtername )

Список параметров

filtername

Имя фильтра.

Возвращаемые значения

ID фильтра в случае успеха или NULL, если фильтр не существует.
