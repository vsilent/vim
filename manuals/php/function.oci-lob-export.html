lob->export — Сохраняет содержимое объекта LOB в файл

Описание

bool ociwritelobtofile ( string $filename [, int $start [, int $length ]] )

Экспортирует содержимое объекта LOB в файл, указанный в параметре filename . Аргумент start определяет с какого места начинать экспортировать данные, а аргумент length - их длину.

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
