sqlite_last_insert_rowid

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_last_insert_rowid — Возвращает идентификатор последней вставленной записи

Описание

int sqlite_last_insert_rowid ( resource $dbhandle )

Возвращает идентификатор последней записи, вставленной в базу данных dbhandle , если таблица, в которую была вставлена запись, содержит поле с авто-инкрементом.

Подсказка

Поля с авто-инкрементом в SQLite объявляются как INTEGER PRIMARY KEY.
