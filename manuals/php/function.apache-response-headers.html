apache_response_headers

(PHP 4 >= 4.3.0, PHP 5)

apache_response_headers — Получить список всех заголовков HTTP ответа

Описание

array apache_response_headers ( void )

Возвращает массив, содержащий все заголовки ответа Apache. Эта функция доступна только для версий PHP >= 4.3.0.

См. также getallheaders() и headers_sent().
