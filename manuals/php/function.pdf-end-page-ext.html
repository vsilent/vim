PDF_end_page_ext

(PECL pdflib:2.0-2.1.3)

PDF_end_page_ext — Finish page

Описание

bool PDF_end_page_ext ( resource $pdfdoc , string $optlist )

Finishes a page, and applies various options. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
