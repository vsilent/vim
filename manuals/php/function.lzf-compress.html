lzf_compress

(PECL lzf:0.1-1.4)

lzf_compress — LZF compression

Описание

string lzf_compress ( string $data )

lzf_compress() compresses the given data .

Список параметров

data

The string to compress.

Возвращаемые значения

Returns the compressed data or FALSE if an error occurred.
