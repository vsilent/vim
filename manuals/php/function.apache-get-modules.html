apache_get_modules

(PHP 4 >= 4.3.2, PHP 5)

apache_get_modules — Получить список загруженных модулей сервера Apache

Описание

array apache_get_modules ( void )

Получить список загруженных модулей сервера Apache.

Возвращаемые значения

Возвращает массив загруженных модулей сервера Apache.

Список изменений

Версия Описание
5.0.0 Стала доступна при использовании PHP в качестве как Apache 1, так и Apache 2 filter API. Ранее это было возможно только для Apache 2 handler API.

Примеры

Пример #1 Пример использования apache_get_modules()

<?php
print_r(apache_get_modules());
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [0] => core
    [1] => http_core
    [2] => mod_so
    [3] => sapi_apache2
    [4] => mod_mime
    [5] => mod_rewrite
)
