rar_close

(No version information available, might be only in CVS)

rar_close — Закрывает Rar архив и высвобождает занятые ресурсы

Описание

bool rar_close ( resource $rar_file )

Закрывает Rar архив и высвобождает все выделенные ранее ресурсы.

rar_close() возвращает FALSE при ошибке.
