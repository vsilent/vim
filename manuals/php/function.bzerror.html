bzerror

(PHP 4 >= 4.3.3, PHP 5, PECL bz2:1.0)

bzerror — Возвращает код и строку ошибки работы с bzip2

Описание

array bzerror ( resource $bz )

Возвращает ассоциативный массив с кодом и строкой ошибки последней bzip2 функции, отработавшей с указателем bz .

Пример #1 Пример использования bzerror()

<?php
$error = bzerror($bz);

echo $error["errno"];
echo $error["errstr"];
?>

См. также bzerrno() и bzerrstr().
