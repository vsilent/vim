PDF_closepath

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_closepath — Close current path

Описание

bool PDF_closepath ( resource $p )

Closes the current path. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
