sinh

(PHP 4 >= 4.0.7, PHP 5)

sinh — Hyperbolic sine

Описание

float sinh ( float $arg )

Returns the hyperbolic sine of arg , defined as (exp(arg) - exp(-arg))/2.

Список параметров

arg

The argument to process

Возвращаемые значения

The hyperbolic sine of arg
