PDF_end_template

(PHP 4 >= 4.0.5, PECL pdflib:1.0-2.1.3)

PDF_end_template — Finish template

Описание

bool PDF_end_template ( resource $p )

Finishes a template definition. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
