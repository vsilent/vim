sqlite_has_more

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_has_more — Определяет, есть ли еще записи в результате запроса

Описание

bool sqlite_has_more ( resource $result )

sqlite_has_more() возвращает TRUE если результат запроса result содержит записи, доступные для выборки, иначе возвращает FALSE.

Смотрите также

sqlite_num_rows() и sqlite_changes().
