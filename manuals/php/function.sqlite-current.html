sqlite_current

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_current — Возвращает текущую запись результата запроса в массиве

Описание

array sqlite_current ( resource $result [, int $result_type [, bool $decode_binary ]] )

sqlite_current() аналогична sqlite_fetch_array() за исключеним того, что sqlite_current() не переходит к следующей записи после выборки данных.

Если текущая позиция в результате запроса находится за последней записью, эта функция возвращает FALSE.

Замечание: Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Смотрите также

sqlite_seek(), sqlite_next() и sqlite_fetch_array().
