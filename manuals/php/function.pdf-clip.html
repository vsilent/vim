PDF_clip

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_clip — Clip to current path

Описание

bool PDF_clip ( resource $p )

Uses the current path as clipping path, and terminate the path. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
