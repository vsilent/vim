PDF_skew

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_skew — Skew the coordinate system

Описание

bool PDF_skew ( resource $p , float $alpha , float $beta )

Skews the coordinate system in x and y direction by alpha and beta degrees. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
