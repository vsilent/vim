rmdir

(PHP 4, PHP 5)

rmdir — Удаляет директорию

Описание

bool rmdir ( string $dirname [, resource $context ] )

Пытается удалить директорию с именем dirname . Директория должна быть пустой и должны иметься необходимые для этого права. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Замечание: Начиная с версии PHP 5.0.0, функция rmdir() также может быть использована с некоторыми обвёртками URL. Обратитесь к List of Supported Protocols/Wrappers для получения списка обвёрток, которые поддерживают rmdir().

Замечание: Аргумент context был добавлен начиная с версии PHP 5.0.0.

Замечание: Когда опция safe mode включена, PHP проверяет, имеет ли каталог, с которым вы собираетесь работать, такой же UID (владельца), как и выполняемый скрипт.

См. также описание функций mkdir() и unlink().
