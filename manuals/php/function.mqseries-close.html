mqseries_close

(PECL mqseries:0.10.0-0.9.0)

mqseries_close — MQSeries MQCLOSE

Описание

mqseries_close ( resource $hconn , resource $hobj , resource $compCode , resource $reason )

The mqseries_close() (MQCLOSE) call relinquishes access to an object, and is the inverse of the mqseries_open() (MQOPEN) call.

Список параметров

hConn

Connection handle.

This handle represents the connection to the queue manager.

hObj

Object handle.

This handle represents the object to be used.

compCode

Completion code.

reason

Reason code qualifying the compCode.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 mqseries_close() example

<?php
    mqseries_close($conn, $obj, MQSERIES_MQCO_NONE, $comp_code, $reason);
    if ($comp_code !== MQSERIES_MQCC_OK) {
        printf("close CompCode:%d Reason:%d Text:%s<br>\n", $comp_code, $reason, mqseries_strerror($reason));
    }
?>
