$_COOKIE

$HTTP_COOKIE_VARS [deprecated]

(No version information available, might be only in CVS)

$HTTP_COOKIE_VARS [deprecated] — HTTP Cookies

Описание

An associative array of variables passed to the current script via HTTP Cookies.

$HTTP_COOKIE_VARS contains the same initial information, but is not a superglobal. (Note that $HTTP_COOKIE_VARS and $_COOKIE are different variables and that PHP handles them as such)

Список изменений

Версия Описание
4.1.0 Introduced $_COOKIE that deprecated $HTTP_COOKIE_VARS.

Примеры

Пример #1 $_COOKIE example

<?php
echo 'Hello ' . htmlspecialchars($_COOKIE["name"]) . '!';
?>

Assuming the "name" cookie has been set earlier

Результатом выполнения данного примера будет что-то подобное:

Hello Hannes!

Примечания

Замечание: This is a 'superglobal', or automatic global, variable. This simply means that it is available in all scopes throughout a script. There is no need to do global $variable; to access it within functions or methods.
