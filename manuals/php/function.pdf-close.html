PDF_close

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_close — Close pdf resource [deprecated]

Описание

bool PDF_close ( resource $p )

Closes the generated PDF file, and frees all document-related resources. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

This function is deprecated since PDFlib version 6, use PDF_end_document() instead.
