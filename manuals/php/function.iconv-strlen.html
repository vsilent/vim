iconv_strlen

(PHP 5)

iconv_strlen — Возвращает количество символов в строке

Описание

int iconv_strlen ( string $str [, string $charset ] )

Возвращает количество символов в str .

В отличие от strlen(), iconv_strlen() учитывает кодировку строки (например, юникод может быть и двух-, и четырёхбайтным).

Если параметр charset опущен, предполагается, что кодировка строки str равна iconv.internal_charset.

См. также strlen() и mb_strlen().
