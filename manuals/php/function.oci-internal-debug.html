oci_internal_debug

(PHP 5, PECL oci8:1.1-1.2.4)

oci_internal_debug — Включает и выключает внутреннюю отладку

Описание

void oci_internal_debug ( int $onoff )

oci_internal_debug() включает внутреннюю отладку модуля OCI8. Укажите значение параметра onoff равное нулю, чтобы выключить отладку и равное единице, чтобы включить её.

Замечание: В версиях PHP ниже 5.0.0 эта функция называлась ociinternaldebug(). В PHP 5.0.0 и выше ociinternaldebug() является алиасом oci_internal_debug(), поэтому вы можете продолжать использовать это имя, однако это не рекомендуется.
