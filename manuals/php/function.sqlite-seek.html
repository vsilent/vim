sqlite_seek

(PHP 5, PECL sqlite:1.0-1.0.3)

sqlite_seek — Переходит к определенному номеру записи

Описание

bool sqlite_seek ( resource $result , int $rownum )

sqlite_seek() перемещает внутренний указатель результата запроса на запись с порядковым номером, заданным аргументом rownum . Нумерация записей начинается с нуля (0 соответствует первой записи). Возвращает FALSE если запись с указанным номером отсутствует, иначе TRUE.

Замечание: Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Смотрите также

sqlite_next(), sqlite_current() и sqlite_rewind().
