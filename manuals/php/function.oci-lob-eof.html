lob->eof

(No version information available, might be only in CVS)

lob->eof — Проверяет, находится ли указатель LOB на конце файла

Описание

boolean lob->eof ( void )

Возвращает TRUE, если внутренний указатель объекта LOB находится на конце файла. В ином случае возвращает FALSE.

См. также oci_lob_size().
