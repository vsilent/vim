PDF_begin_page_ext

(PECL pdflib:2.0-2.1.3)

PDF_begin_page_ext — Start new page

Описание

bool PDF_begin_page_ext ( resource $pdfdoc , float $width , float $height , string $optlist )

Adds a new page to the document, and specifies various options. The parameters width and height are the dimensions of the new page in points. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
