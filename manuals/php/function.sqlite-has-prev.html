sqlite_has_prev

(PHP 5)

sqlite_has_prev — Определяет, доступна ли предыдущая запись

Описание

bool sqlite_has_prev ( resource $result )

sqlite_has_prev() возвращает TRUE если предыдущая запись результата запроса result доступна, иначе возвращает FALSE.

Замечание: Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Смотрите также

sqlite_prev(), sqlite_has_more() и sqlite_num_rows().
