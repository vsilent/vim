PDF_activate_item

(PECL pdflib:2.0-2.1.3)

PDF_activate_item — Activate structure element or other content item

Описание

bool PDF_activate_item ( resource $pdfdoc , int $id )

Activates a previously created structure element or other content item. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
