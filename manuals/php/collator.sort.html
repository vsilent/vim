Collator::sort

collator_sort

(No version information available, might be only in CVS)

collator_sort — Sort array using specified collator

Описание

Object oriented style

bool Collator::sort ( array &$arr [, integer $sort_flag ] )

Procedural style

bool collator_sort ( Collator $coll , array &$arr [, integer $sort_flag ] )

This function sorts an array according to current locale rules.

Equivalent to standard PHP sort() .

Список параметров

coll

Collator object.

arr

Array of strings to sort.

sort_flag

Optional sorting type, one of the following:

  • Collator::SORT_REGULAR - compare items normally (don't change types)

  • Collator::SORT_NUMERIC - compare items numerically

  • Collator::SORT_STRING - compare items as strings

Default sorting type is Collator::SORT_REGULAR.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 collator_sort() example

<?php
$coll = collator_create( 'en_US' );
$arr  = array( 'at', 'às', 'as' );

var_export( $arr );
collator_sort( $coll, $arr );
var_export( $arr );
?>

Результат выполнения данного примера:

array (
  0 => 'at',
  1 => 'às',
  2 => 'as',
)array (
  0 => 'as',
  1 => 'às',
  2 => 'at',
)
