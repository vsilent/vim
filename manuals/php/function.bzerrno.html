bzerrno

(PHP 4 >= 4.3.3, PHP 5, PECL bz2:1.0)

bzerrno — Возвращает код ошибки работы с bzip2

Описание

integer bzerrno ( resource $bz )

Возвращает код ошибки последней bzip2 функций, отработавшей с указателем bz .

См. также bzerror() и bzerrstr().
