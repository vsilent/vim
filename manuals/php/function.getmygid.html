getmygid

(PHP 4 >= 4.0.7, PHP 5)

getmygid — Get PHP script owner's GID

Описание

int getmygid ( void )

Gets the group ID of the current script.

Возвращаемые значения

Returns the group ID of the current script, or FALSE on error.
