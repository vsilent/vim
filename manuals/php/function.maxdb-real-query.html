maxdb_real_query

maxdb->real_query

(PECL maxdb:1.0-7.6.00.38)

maxdb->real_query — Execute an SQL query

Описание

Procedural style

bool maxdb_real_query ( resource $link , string $query )

Object oriented style (method):

maxdb
bool real_query ( string $query )

The maxdb_real_query() is functionally identical with the maxdb_query().

Замечание: In order to determine if a given query should return a result set or not, see maxdb_field_count().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
