PDF_scale

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_scale — Scale coordinate system

Описание

bool PDF_scale ( resource $p , float $sx , float $sy )

Scales the coordinate system. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
