SplStack::__construct

(No version information available, might be only in CVS)

SplStack::__construct — Constructs a new stack implemented using a doubly linked list

Описание

SplStack::__construct ( void )

This constructs a new empty stack.

Замечание: This method automatically sets the iterator mode to SplDoublyLinkedList::IT_MODE_LIFO.

Список параметров

This function has no parameters.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 SplStack::__construct() example

<?php
$q = new SplStack();

$q[] = 1;
$q[] = 2;
$q[] = 3;

foreach ($q as $elem)  {
 echo $elem."\n";
}
?>

Результат выполнения данного примера:

3
2
1
