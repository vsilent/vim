PDF_end_page

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_end_page — Finish page

Описание

bool PDF_end_page ( resource $p )

Finishes the page. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
