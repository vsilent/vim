HttpResponse::setThrottleDelay

(PECL pecl_http:0.10.0-1.5.5)

HttpResponse::setThrottleDelay — Set throttle delay

Описание

static bool HttpResponse::setThrottleDelay ( float $seconds )

Sets the throttle delay.

Замечание: Provides a basic throttling mechanism, which will yield the current process or thread until the entity has been completely sent.

Замечание: This may not work as expected with the following SAPI(s): FastCGI.

Список параметров

seconds

seconds to sleep after each chunk sent

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
