call_user_method_array

(PHP 4 >= 4.0.5, PHP 5)

call_user_method_array — Вызываем метод объекта с массивом параметров [устаревшее]

Описание

mixed call_user_method_array ( string $method_name , object $obj [, array $paramarr ] )
Внимание

Функция call_user_method_array() устарела с выходом 4.1.0: используйте функцию call_user_func_array() с array(&$obj, "method_name") в качестве параметра имени функции.

Вызывает метод method_name объекта obj , с параметрами содержащимися в массиве paramarr .

См. также: call_user_func_array() и call_user_func().
