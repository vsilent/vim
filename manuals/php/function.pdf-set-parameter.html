PDF_set_parameter

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_set_parameter — Set string parameter

Описание

bool PDF_set_parameter ( resource $p , string $key , string $value )

Sets some PDFlib parameter with string type. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
