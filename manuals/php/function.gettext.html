gettext

(PHP 4, PHP 5, PECL axis2:0.1.0-0.1.1)

gettext — Возвращает перевод сообщения

Описание

string gettext ( string $message )

Возвращает перевод сообщения, найденный в текущем каталоге сообщений, или оригинал, если перевод не найден. Псевдоним функции - '_'.

Пример #1 gettext()-check

<?php
// Установить язык в русский
setlocale(LC_ALL, 'ru_RU');

// Указать путь к каталогу сообщений
bindtextdomain("myPHPApp", "./locale");

// Выбрать домены
textdomain("myPHPApp");

// Теперь переводы будут искаться в ./locale/de_DE/LC_MESSAGES/myPHPApp.mo

// Вывести текстовое сообщение
echo gettext("Welcome to My PHP Application");

// То же самое
echo _("Have a nice day");
?>

См. также setlocale().
