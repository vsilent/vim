imagegd

(PHP 4 >= 4.0.7, PHP 5)

imagegd — Output GD image to browser or file

Описание

bool imagegd ( resource $image [, string $filename ] )

Outputs a GD image to the given filename .

Список параметров

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

filename

The path to save the file to. If not set or NULL, the raw image stream will be outputted directly.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примечания

Замечание: The GD format is commonly used to allow fast loading of parts of images. Note that the GD format is only usable in GD-compatible applications.
