bzerrstr

(PHP 4 >= 4.3.3, PHP 5, PECL bz2:1.0)

bzerrstr — Возвращает строку ошибки работы с bzip2

Описание

string bzerrstr ( resource $bz )

Возвращает строку ошибки последней bzip2 функции, отработавшей с указателем bz .

См. также bzerrno() and bzerror().
