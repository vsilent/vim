rar_list — Возвращает список элементов, входящих в Rar архив

Описание

array rar_list ( resource $rar_file )

Возвращает список элементов, входящих в Rar архив.

Пример #1 Rar::() пример использования

<?php

$rar_file = rar_open('example.rar') or die("Невозможно открыть Rar архив");

$entries_list = rar_list($rar_file);

print_r($entries_list);

?>

rar_list() возвращает массив элементов архива или FALSE при ошибке.
