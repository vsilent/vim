mb_http_output

(PHP 4 >= 4.0.6, PHP 5)

mb_http_output — Set/Get HTTP output character encoding

Описание

mixed mb_http_output ([ string $encoding ] )

Set/Get the HTTP output character encoding. Output after this function is converted to encoding .

Список параметров

encoding

If encoding is set, mb_http_output() sets the HTTP output character encoding to encoding .

If encoding is omitted, mb_http_output() returns the current HTTP output character encoding.

Возвращаемые значения

If encoding is omitted, mb_http_output() returns the current HTTP output character encoding. Otherwise, Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
