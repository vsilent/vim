ctype_graph

(PHP 4 >= 4.0.4, PHP 5)

ctype_graph — Проверяет, является ли указанный символ печатаемым, за исключением пробельного

Описание

bool ctype_graph ( string $text )

Проверяет, все ли символы в переданной строке text являются видимыми при печати.

Список параметров

text

Проверяемая строка.

Возвращаемые значения

Возвращает TRUE, если каждый символ в строке text печатаемый и фактически создает видимый вывод (без пробельных символов), FALSE в противном случае.

Примеры

Пример #1 Пример использования ctype_graph()

<?php
$strings = array('string1' => "asdf\n\r\t", 'string2' => 'arf12', 'string3' => 'LKA#@%.54');
foreach ($strings as $name => $testcase) {
    if (ctype_graph($testcase)) {
        echo "Строка '$name' состоит только из (видимых) печатаемых символов.\n";
    } else {
        echo "Строка '$name' не состоит только из (видимых) печатаемых символов.\n";
    }
}
?>

Результат выполнения данного примера:

Строка 'string1' не состоит только из (видимых) печатаемых символов.
Строка 'string2' состоит только из (видимых) печатаемых символов.
Строка 'string3' состоит только из (видимых) печатаемых символов.
