PDF_set_border_style

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_set_border_style — Set border style of annotations [deprecated]

Описание

bool PDF_set_border_style ( resource $pdfdoc , string $style , float $width )

Sets the border style for all kinds of annotations. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

This function is deprecated since PDFlib version 6, use the options borderstyle and linewidth in PDF_create_annotation() instead.
