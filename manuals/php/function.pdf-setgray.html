PDF_setgray

(PHP 4, PECL pdflib:1.0-2.1.3)

PDF_setgray — Set color to gray [deprecated]

Описание

bool PDF_setgray ( resource $p , float $g )

Sets the current fill and stroke color to a gray value between 0 and 1 inclusive. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

This function is deprecated since PDFlib version 4, use PDF_setcolor() instead.
